本文的环境为ubuntu14.04.4LTS g++4.8.4 python2.7

##安装boost和python-dev

boost可以使用apt-get安装
sudo apt-get install libboost-all-dev
安装完成之后目录在/usr/include/boost下，且安装程序以及将boost的路径添加到了PATH中了
python-dev依然可以使用apt-get安装
sudo apt-get install python-dev
